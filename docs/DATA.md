# Data sources and access

This repository contains analysis code only. The underlying data are not redistributed because of size and licensing. This document lists each dataset, its source, and how to obtain a copy.

## Multi temporal G LiHT airborne LiDAR (2012, 2014, 2015, 2017, 2021, 2023, 2025)

NASA Goddard's LiDAR, Hyperspectral and Thermal Imager (G LiHT) campaigns over Howland Research Forest. Distributed via ORNL DAAC.

- 2012, 2017, 2021 normalized point clouds: `data/gliht_pointclouds/ME_<YYYYMMDD>_Howland_Forest/`
- 2025 MELiTE acquisition (Phoenix RANGER, ~92 pts m⁻²): `data/als_2025/` (raw) and `output/als_2025_processing/` (normalized tiles)
- 25 pts m⁻² decimated tiles: `output/als_2025_decimated_25/` (cached for Method 10 reruns)

Access:
- ORNL DAAC: https://daac.ornl.gov/ search "G LiHT Howland"
- Raw 2025 acquisition: contact NASA G LiHT mission

## ORNL G LiHT AGB rasters

Per pixel AGB carbon density derived from G LiHT processing. Used as anchor for Method 9 stratification, AGB trajectory comparison, and the Method 11/12 ORNL sensitivity test.

- `data/ornl_agb_rasters/Howland_AGB_<YYYY>.tif` for 2012, 2015, 2017, 2021, 2023, 2025

Access: ORNL DAAC product search.

## 2025 MELiTE ABA AGB raster (validation reference)

10 m gridded area based AGB derived from the 2025 MELiTE acquisition using the Howland recalibrated p95 model: AGB = 1.572 × p95^1.528. This raster is the validation target for Methods 9 through 12.

- `output/agb_2025_melite_30m.tif` (despite the filename, native resolution is 10 m)

Generated by the recalibration pipeline; file is provided in the analysis archive on Cardinal under project allocation PUOM0008.

## 2025 terrestrial laser scanning

RIEGL VZ 400i campaign covering 29 strata at Howland. 4,417 segmented stems, 3,398 high quality (DBH confidence >= 50). Provides empirical bounds on segmentation and aggregation error and the H D + crown radius calibration source for Method 12.

- `data/tls_2025/Merged_data_total.txt`
  - Columns: X, Y, Z, TileID, TreeID, GlobalTreeID, Height (m), DBH (cm), confidence_location, confidence_DBH

Access: Purdue forestry remote sensing group (contact Purdue collaborators).

## 2017 wall to wall ITC tree list

Individual tree crown segmentation of the 2017 G LiHT acquisition. 208,632 trees with treeID, Z (height), crown_radius_m, crown_area_m2 in 3D point geometry (UTM 19N).

- `output/Howland_ITC_2017.gpkg` (155 MB)

Generated by the ITC pipeline at `scripts/03_howland_individual_tree_efi_fvs.R` (not in this repository; available on Cardinal under PUOM0008). Used as starting state for Methods 11 and 12.

## Field inventory plots

56 fixed area ground plots with tree level DBH, height, species, and snag status. Used for stratification, species composition assignment, and field reference validation.

- `data/field_inventory/Plots_Data.csv` (plot center coordinates, area, radius)
- `data/field_inventory/Tree_Data.csv` (per tree records)

Access: Howland Research Forest data archive.

## AcadianGY v12.3.6

R based individual tree growth and yield model for Acadian forest types in the Northeast.

- `AcadianGY_12.3.6.r` (sourced as the AcadianGYOneStand engine)

Access: Weiskittel lab, University of Maine.

## Compute allocation

All code was developed and run on the Ohio Supercomputer Center (OSC) Cardinal cluster under project PUOM0008. Cardinal users with access to that allocation can find all data, intermediate RDS files, and figure outputs at:

```
/users/PUOM0008/crsfaaron/HRF/
├── data/                # raw inputs
├── output/              # all derived products
└── scripts/             # canonical analysis scripts
```

Non Cardinal users will need to substitute their own paths and module loads.
